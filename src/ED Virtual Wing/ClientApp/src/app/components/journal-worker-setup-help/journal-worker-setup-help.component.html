Journal Setup

First, open the main directory of your current user, as shown in the screenshot above. In this case, the user name is Admin.



Select the folder Saved Games.



Select the folder Frontier Developments.



Select the folder Elite Dangerous.



Click on the button Select Folder.



The setup is now complete. Every time when you start sharing your game logs, a prompt from your browser will appear.
Once you confirm it with View files the virtual team application will be able to read
your game journal and share your game activities with your wing(s).
